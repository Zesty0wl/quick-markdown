# Support

Thanks for using Quick Markdown.

## Where to ask for help

- Usage questions: open a GitHub Discussion (if enabled) or issue
- Bug reports: use the Bug Report issue template
- Feature ideas: use the Feature Request issue template
- Security issues: follow SECURITY.md and report privately

## Before opening an issue

Please include:

- macOS version
- Quick Markdown version or commit SHA
- Steps to reproduce
- Expected behavior
- Actual behavior
- A minimal markdown sample that reproduces the problem

For rendering or formatting bugs, small markdown examples are the fastest way to
help us reproduce and fix the issue.
