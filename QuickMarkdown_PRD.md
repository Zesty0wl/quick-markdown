# Quick Markdown Product Scope

## Product statement

Quick Markdown is a focused native macOS app for opening, editing, and sharing
one markdown file at a time.

## Target users

- People drafting markdown with AI assistants
- Engineers writing README and technical docs
- Writers and knowledge workers who want clean markdown plus rich export

## Core jobs to be done

1. Open markdown files quickly and render them beautifully.
2. Edit with minimal friction while preserving source fidelity.
3. Keep up with external file changes in near real time.
4. Share content through rich clipboard and PDF export.

## Core principles

- Source markdown remains the system of record.
- Native macOS behavior over framework abstraction.
- Narrow scope beats feature sprawl.
- Fast and predictable interactions over complex customization.

## In scope

- Markdown editing and live preview for a single document
- Source mode and formatting tools
- Rich clipboard output (HTML, RTF, plain text)
- PDF export
- Accessibility and reading comfort features

## Out of scope

- Multi-file project workspaces
- Plugin marketplace or scripting platform
- Cloud sync, collaboration, or account features
- Notes database or vault model
- Non-markdown authoring formats as first-class targets

## Quality bar

- Reliable opening/saving of markdown files
- Stable behavior during external file writes
- Keyboard-first editing and formatting workflows
- Predictable export output in mainstream document apps
