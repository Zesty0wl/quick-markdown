# Security Policy

## Supported Versions

Quick Markdown is developed on the main branch, with tagged releases supported
as noted below.

- main: supported
- tagged releases from 1.0.0 onward: supported until superseded by a newer
  minor release

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Use one of these private channels:

1. GitHub Security Advisories for this repository (preferred)
2. Email: neil@zestyowl.com

Please include:

- A description of the issue and impact
- Reproduction steps or a proof of concept
- Affected version, commit, or branch
- Any suggested mitigation

### What to expect

- Initial response target: within 3 business days
- Triage/update target: within 7 business days
- Coordinated disclosure after a fix is available

We will do our best to acknowledge valid reports quickly and keep reporters
informed throughout triage and remediation.
